## 📄 Overview

This project is a personal portfolio website designed and developed from scratch using only HTML and CSS. It includes multiple navigable sections, each dedicated to a specific area of interest or information, and serves as a comprehensive, interactive digital resume.

---

## ✨ Key Highlights

* 🎨 **Minimal and Modern Design**
* 📱 **Responsive Layout** – Works on mobile, tablet, and desktop
* 📁 **Multi-Page Structure** – For better content organization
* 📷 **Photography Gallery** – To showcase creative work
* 📄 **Online Resume Link**
* 🌐 **Social Media Integration** – LinkedIn, GitHub, Email

---

## 🧭 Navigation Structure

Your portfolio includes the following pages:

| Page               | Description                                     |
| ------------------ | ----------------------------------------------- |
| `index.html`       | Home page with name, greeting, and social links |
| `about.html`       | Personal introduction and background            |
| `education.html`   | Educational qualifications                      |
| `projects.html`    | Details of selected technical projects          |
| `skills.html`      | Technical and soft skills overview              |
| `photography.html` | Photography gallery showcasing visual work      |

## 🚀 How to Use

To run the website locally:

1. **Clone the repository**

   ```bash
   git clone https://github.com/SankalpDev/Personal-portfolio.git
   ```

2. **Navigate into the project directory**

   ```bash
   cd Personal-portfolio
   ```

3. **Open `index.html` in your browser**

   * Double-click the file **or**
   * Use a development server like VS Code’s Live Server for dynamic preview

4. **Customize content** in the HTML pages or modify styles in `styles.css` as per your preference.
